2013/0417Wed白石麻衣ジリリリリ(o^_^o)

こんばんは。
名古屋個別ありがとうです(^_−)−☆




1部2部5部は、黒のワンピース

あ...足元スリッパなのはごめんなさい(/ _ ; )




2部でセーラームーンみたいなツインテールをしました。(前に貼ったロリータメイクの時みたいな髪型。詳しくは...そのうち更新される松村ブログまで笑)


恥ずかしかったけど、意外と好きって言ってくださった方がたくさんいたので嬉しかったなぁ(*^◯^*)


3部4部はマイオをやりました。

いこたんと♡

きっとマイオはこれが見納めです泣




口元には、メイクで殴られた跡を作ったんだけど、本気で心配してくれた人がいて...なんかすみませんでした(/ _ ; )


シャキイズムのPVの時の格好です！
まだ見てない方は是非とも見て下さい(^-^)/






乃木どこ？の途中で流れた、フォーチュンミュージックのCM、見られた方いますか？


個別握手会の券を購入するサイトが、今後からフォーチュンミュージックと言うサイトに変わるのですが...そのCMです！

曖昧でごめんなさい笑



拳法みたいな動きをしているのですが、「3・2・1」でポーズを変えて握手をするという動きを30回?くらいやって、普段しない動きをした為、足が筋肉痛になりました。


前ブログ言ってた足が筋肉痛〜の理由は、これのことです(；´・_・｀)ゞ


剥がし役の方もいるので、注目してみてね！笑






メイドーん。その二




明日は早起きだーーー





まいやん。
