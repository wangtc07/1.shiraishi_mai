2012/0228Tue白石麻衣白い肌がチャームポイント(o^_^o)

こんばんは(^ ^)
 
 
 
遅くに更新ごめんね（ ;  ; ）
 
 
 
 
レッスンわずです！
 
 
 
 
 
あれれ～？
 
 
二人ニセモノが混ざってる！
 
 
 
 
 
 
みなみがさゆりんごで、いくちゃんがなーちゃんです！笑
 
 
 
いくちゃんのブログに貼ってあった画像を真似して撮ってみました。
 
 
いくちゃん、ごめんね（ ;  ; ）
 
 
 
そして、北海道版のななみんはまだレッスンに来てなかったので一緒に撮れませんでした（ ;  ; ）泣
 
 
 
 
はい！
今回、BLTさんの表紙を私達が飾らせてもらいました(*^^*)
 
それと、裏表紙も乃木坂46がジャック！
 
 
 
私は九州版の方に載っています(^ ^)
 
 
なので是非、九州の皆さんお時間があったらチラッと見て頂けたら嬉しいです☆
 
 
 
 
 
話変わりまして、4日間のミニ握手会が終わりました！
 
私は成田、名古屋、草津、高岡の方にお邪魔しました。
 
 
たくさんの方に『デビューおめでとう』などの、暖かい声援をもらって本当に嬉しかったですし、デビューできたのも、ずっと応援してくださったファンの皆さんのおかげです。本当に感謝しています！
 
 
 
 
まだまだ乃木坂46は始まったばかりですが、今、目の前にある事を一つ一つ乗り越えて目標に向かって日々、少しずつ前進していくので、これからも応援よろしくお願いします！
 
 
 
 
うちらは乃木坂 上り坂！
 
 
 
 
 
3月3日には、始めての全国握手会もあるので、まずはそれに向けて一人一人が頑張っていきたいと思います(*^^*)
 
 
 
早く皆さんに会いたくて仕方がないです。
 
 
全力で頑張るので、一緒に楽しもうね
 
 
 
 
 
明日もお仕事なので、おわり☆
 
 
 
 
 
 
まいやん。
